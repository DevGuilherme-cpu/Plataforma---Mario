# 🍄 Super Pygame Adventure

## 📜 Descrição do Jogo
**Super Pygame Adventure** é um jogo de plataforma 2D no estilo clássico (Platformer), desenvolvido inteiramente com a biblioteca Pygame Zero (PgZero). O projeto foi criado como avaliação para a disciplina de Linguagem de Programação da Faculdade Serra do Carmo.

## 🎮 Controles
O jogo possui controles simples e intuitivos:
* **Menu Principal:** Utilize o **Mouse** (Clique Esquerdo) para interagir com os botões (Iniciar, Áudio e Sair).
* **Movimentação:** Pressione as setas **Esquerda** ($\leftarrow$) e **Direita** ($\rightarrow$) do teclado para andar.
* **Pulo:** Pressione a seta para **Cima** ($\uparrow$) para pular.
* **Telas de Fim de Jogo (Game Over / Vitória):** Pressione **ENTER** para reiniciar a partida.
* **Sair da Fase:** Pressione **ESC** durante o jogo para voltar ao Menu Principal.

## 🚀 Como Executar

Para rodar este jogo em sua máquina, certifique-se de ter o Python instalado. Siga os passos abaixo:

1. **Instale a biblioteca Pygame Zero:**
   Abra o seu terminal (ou o terminal do VS Code) e digite o seguinte comando:
   `pip install pgzero`

2. **Organização das Pastas:**
   Certifique-se de que o arquivo `main.py` está na pasta principal e que as pastas de recursos (`images`, `sounds` e `music`) estão no mesmo diretório, contendo os respectivos arquivos de mídia.

3. **Iniciando o Jogo:**
   Pelo terminal, navegue até a pasta do projeto e execute o comando:
   `pgzrun main.py`
   
   *(Observação: Se estiver usando o VS Code, você também pode simplesmente executar o arquivo `main.py` normalmente, pois o script já possui a chamada `pgzrun.go()` no final do código).*
